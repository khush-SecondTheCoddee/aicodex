# Personal Page (Next.js)

This is a simple Next.js personal page that reads all profile data from `myinfo.json`.

## Customize your data

Edit the `myinfo.json` file to update your:

- Name, role, location, bio, email
- Avatar image URL
- Skills
- Social links
- Projects

Whenever you change `myinfo.json`, your page content updates accordingly.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
